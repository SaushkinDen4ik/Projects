# Установка необходимых пакетов:
В данной игре требуется установка пакета kivy, делается это следующим образом:
Для пользователй Linux необходимо прописать в консоли: 
python -m pip install kivy
или же
python3 -m pip install kivy
в зависимости от версии python на вашем компьютере.

# Запуск + Функционал
Запустив данную программу появится окно с тремя кнопками.
Чтобы получать очки требуется прожимать на пробел, данные очки можно (и нужно) расходовать на улучшение: Level up (ваш уровень отображается в левой части окна) + увеличение числа очков за каждый клик.
Кликайте больше, вкачивайте уровни и удачи!
