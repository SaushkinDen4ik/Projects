# Функционал сайта:
Создание поста, возможность его просмотра, удаления. 
Возможность оставлять фидбэк, тогда соответсвующий текст будет отображаться в базе данных tablefeedback  
Также есть страница с описанием создателя и информацией о сайте (о источниках вдохновения также)  
(Есть секретная возможность для удобства чистки базы данных удалить все посты перейдя по URL /removeall)  
Градиентный фон для красоты ^_^

# Как запускать (запуск локально на своём компьютере):
0) Скачать репозиторий git clone git@github.com:SaushkinDen4ik/Projects.git
1) Перейти в папку flask_web_service
2) Прописываем в консоли
3) pip install flask
4) pip install virtualenv
5) virtualenv env
6) source env/bin/activate
7) export FLASK_APP=digit
8) export FLASK_ENV=development
9) flask run (-p 'номер порта', если хотите на другом порту запустить)
10) Скопировать URL и перейти по нему
